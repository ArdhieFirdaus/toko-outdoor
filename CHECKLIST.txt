================================================================================
TROUBLESHOOTING GUIDE
================================================================================

Problem: "Gagal terkoneksi ke database"
Solution:
  1. Ensure MySQL is running in XAMPP
  2. Check credentials in Config/koneksi.php
  3. Verify database name is "toko_outdoor"
  4. Run test_connection.php to debug

Problem: "Session expired" or cannot login
Solution:
  1. Clear browser cookies
  2. Try in incognito/private mode
  3. Restart XAMPP
  4. Check if koneksi.php is in correct location

Problem: Stok tidak berkurang
Solution:
  1. Verify transaksi created successfully
  2. Check detail_transaksi table in database
  3. Ensure quantity is valid in form
  4. Clear cache and try again

Problem: CSS tidak load / tampilan berantakan
Solution:
  1. Check assets/css/style.css exists
  2. Verify browser cache (Ctrl+F5)
  3. Check browser console for 404 errors
  4. Ensure .htaccess is in root folder
