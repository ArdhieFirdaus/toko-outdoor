TROUBLESHOOTING
================================================================================

‚ùå MASALAH: Tidak bisa login
   ‚úÖ SOLUSI:
      1. Cek username & password sesuai (admin/admin123)
      2. Pastikan database sudah di-import
      3. Clear cookies browser (Ctrl+Shift+Delete)
      4. Buka file: test_connection.php untuk debug

‚ùå MASALAH: Error "Gagal terkoneksi ke database"
   ‚úÖ SOLUSI:
      1. Pastikan MySQL sudah running di XAMPP
      2. Cek file: Config/koneksi.php
      3. Jalankan test_connection.php
      4. Pastikan database bernama "toko_outdoor" ada

‚ùå MASALAH: CSS tidak load / tampilan berantakan
   ‚úÖ SOLUSI:
      1. Refresh browser: Ctrl+F5 (hard refresh)
      2. Cek file: assets/css/style.css ada
      3. Buka browser console (F12) cek error
      4. Pastikan folder assets ada

‚ùå MASALAH: Stok tidak berkurang saat transaksi
   ‚úÖ SOLUSI:
      1. Cek apakah transaksi berhasil dibuat
      2. Login sebagai admin ‚Üí lihat database
      3. Pastikan quantity input valid (> 0)
      4. Refresh halaman (F5)

‚ùå MASALAH: Session expired
   ‚úÖ SOLUSI:
      1. Clear cookies browser
      2. Tutup browser sepenuhnya
      3. Buka browser baru
      4. Login kembali
